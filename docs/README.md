# Documentation

Reference docs for the Vircon32 synthesizer library (`synth.h`).

- **[concepts.md](concepts.md)** — how the synth works, what your ROM needs, and
  the hardware facts that shape what's possible. **Start here.**
- **[api-reference.md](api-reference.md)** — every function, type, and constant.

New to the library? Follow the [tutorials](../tutorials/) first — they walk you
from a single sound effect to a full soundtrack and custom instruments. Then use
these docs as reference.

See also the top-level [README](../README.md) (overview + quick start) and
[CLAUDE.md](../CLAUDE.md) (contributor/engine internals).
